# Project Title: Cyber Task Tracker

## Project Description:
A simple program that tracks tasks that the user inputs. It sets deadlines based on the user's preference.

## Features:
- Input tasks
- Input deadlines for tasks
- Sets a table for deadlines and tasks
- Input task-specific instructions
  e.g.: Sweep the floor
        Step 1: grab the broom
        Step 2: sweep the floor
- Other users can also input tasks for you if you accept

## How to run the program:
- Make sure you have python on your device
- download the file "task_tracker.py"
- Open terminal or command prompt to run the program
- Run the program by pressing F5 or clicking run
- Follow the instructions set
